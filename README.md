# HANGAKOBO: MOMOIRO NEKO

版画家おさのなおこの出版「ももいろのねこ」をインタラクティブWeb絵本として制作しました。  
シンプルな構成で、ページめくりの演出を試験的に実装しています。  
JavaScriptによるフルスクラッチで構築された SPA 作品です。
